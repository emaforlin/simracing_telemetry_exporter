# simracing_telemetry_exporter

This project is a telemetry exporter for simracing games, capable of reading telemetry data from multiple simulation games and exporting it to a serial port.

## Features

- Read telemetry data from multiple simracing games
- Export telemetry data to a serial port

## Installation

```bash
git clone https://github.com/emaforlin/simracing_telemetry_exporter.git
cd simracing_telemetry_exporter
cmake -S . -B build
cmake --build build
```

## Usage

```bash
./build/simracing_telemetry_exporter
```
